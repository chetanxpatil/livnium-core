# Grid & distances

The lattice uses coordinates in {-1,0,1}^3:

```dart
Iterable<Vec3> cube3Coords() sync* {
  for (var x = -1; x <= 1; x++) {
    for (var y = -1; y <= 1; y++) {
      for (var z = -1; z <= 1; z++) {
        yield Vec3(x, y, z);
      }
    }
  }
}
```

Manhattan distance helper:

```dart
int l1(Vec3 v) => v.x.abs() + v.y.abs() + v.z.abs();
```
